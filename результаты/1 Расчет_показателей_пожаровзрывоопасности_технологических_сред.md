# Расчет показателей пожаровзрывоопасности технологических сред

## Введение

Настоящий расчет выполнен в соответствии с Методикой определения показателей пожаровзрывоопасности технологических сред (далее - Методика [1]) и в соответствии с требованиями ГОСТ 12.1.044-2018 "Система стандартов безопасности труда. Пожаровзрывоопасность веществ и материалов. Номенклатура показателей и методы их определения".

## Объект исследования

Объектом исследования является устьевая нефть из скважеры 405Р Орехово-Ермаковского месторождения, далее - "нефть устьевая".

## Расчет основных показателей пожаровзрывоопасности

### 1. Определение класса пожаровзрывоопасности технологической среды

#### 1.1. Классификация по агрегатному состоянию

Технологическая среда представляет собой:
- Жидкость - нефть устьевая
- Температура вспышки: 35°C
- Поскольку температура вспышки < 61°C, вещество относится к легковоспламеняющимся жидкостям (ЛВЖ) согласно ГОСТ 12.1.044-2018

#### 1.2. Расчет концентрационных пределов воспламенения

Для углеводородов с химической формулой C₁₅H₃₂:

Нижний концентрационный предел воспламенения (НКПР):
- По справочным данным: 1,4% (об.)

Верхний концентрационный предел воспламенения (ВКПР):
- Для углеводородов с числом атомов углерода > 6: ВКПР = 3,2 · НКПР
- ВКПР = 3,2 · 1,4 = 4,48% (об.)

#### 1.3. Расчет температуры самовоспламенения

Для углеводородов по формуле (8) из Методики [1]:
T_св = 750 + 6,3 · n_C - 3,47 · n_C² + 0,126 · n_C³ - 0,0021 · n_C⁴

Где n_C = 15 (количество атомов углерода в молекуле C₁₅H₃₂)

T_св = 750 + 6,3 · 15 - 3,47 · 15² + 0,126 · 15³ - 0,0021 · 15⁴
T_св = 750 + 94,5 - 780,75 + 425,25 - 10,63
T_св = 478,37°C

Принимаем температуру самовоспламенения: 480°C

#### 1.4. Расчет минимальной энергии зажигания

Минимальная энергия зажигания рассчитывается по формуле (7) из Методики [1]:
E_min = 0,23 · exp(-0,25 · C_н/C_стех)

Для углеводородов стехиометрическая концентрация:
C_стех = 100/[1 + n_O(1 + 3,76)]

Реакция горения: C₁₅H₃₂ + 23O₂ → 15CO₂ + 16H₂O
n_O = 23 молей кислорода на 1 моль вещества

C_стех = 100/[1 + 23(1 + 3,76)] = 100/[1 + 23·4,76] = 100/[1 + 109,48] = 0,91% (об.)

E_min = 0,23 · exp(-0,25 · 1,4/0,91) = 0,23 · exp(-0,385) = 0,23 · 0,681 = 0,157 мДж

#### 1.5. Расчет максимального давления взрыва

По формуле (8) из Методики [1]:
P_max = P₀ · [1 + (T_b · Δn)/T₀]

Где:
- P₀ = 101,3 кПа (начальное давление)
- T_b = 200 K (температура продуктов сгорания для углеводородов)
- T₀ = 298 K (начальная температура)
- Δn = 1 (изменение числа молей при сгорании 1 моля горючего вещества)

P_max = 101,3 · [1 + (2200 · 1)/298] = 101,3 · [1 + 7,38] = 101,3 · 8,38 = 849,9 кПа

#### 1.6. Расчет скорости нарастания давления при взрыве

По формуле (9) из Методики [1]:
(dP/dt)_max = K_st · P_max · V^(-1/3)

Для углеводородов K_st = 50 с⁻¹, при V = 1 м³:
(dP/dt)_max = 50 · 849,9 · 1^(-1/3) = 42 495 кПа/с

### 2. Оценка условий образования опасных смесей

#### 2.1. Время испарения нефти

При температуре 15-18°C (указано в ТЗ) и при разливе нефти:
- Удельная массовая скорость выгорания: 0,025 кг/(с·м²) = 0,025 кг/(с·м²) = 250 кг/(с·м²)

#### 2.2. Оценка концентрации паров при аварийной ситуации

Для помещения площадью 34,3 м² (помещение 3.66 по ТЗ) и высотой 2,7 м:
- Объем помещения: V = 34,3 · 2,7 = 92,61 м³

При разливе 430 литров нефти (из ТЗ):
- Масса нефти: m = 430 · 0,86 = 369,8 кг
- Плотность нефти при 20°C: 860 кг/м³

## Определение группы опасности технологической среды

Согласно Методике [1], технологические среды пожаровзрывоопасности подразделяются на следующие группы:

1. **Пожароопасные** - если возможно образование горючей среды, а также появление источника зажигания достаточной мощности для возникновения пожара.

2. **Пожаровзрывоопасные** - если возможно образование смесей окислителя с горючими газами, парами легковоспламеняющихся жидкостей, горючими аэрозолями и горючими пылями, в которых при появлении источника зажигания возможно инициирование взрыва и (или) пожара.

3. **Взрывоопасные** - если возможно образование смесей воздуха с горючими газами, парами легковоспламеняющихся жидкостей, горючими жидкостями, горючими аэрозолями и горючими пылями или волокнами и если при определенной концентрации горючего и появлении источника инициирования взрыва (источника зажигания) она способна только взрываться.

4. **Пожаробезопасные** - пространство, в котором отсутствуют горючая среда и (или) окислитель.

### Оценка по критериям:

1. Нефть устьевая - легковоспламеняющаяся жидкость (температура вспышки 35°C < 61°C)
2. НКПР = 1,4% (об.) - ниже нормативного порога
3. Температура самовоспламенения = 480°C - выше комнатной температуры
4. Минимальная энергия зажигания = 0,157 мДж - достаточно низкая, что делает вещество чувствительным к зажиганию

При аварийной ситуации (разлив нефти) возможно образование взрывоопасных концентраций паров в воздухе.

**Вывод:** Технологическая среда (нефть устьевая) относится к **пожаровзрывоопасной** группе (Группа 2) согласно статье 16 Федерального закона № 123-ФЗ, так как:
- Возможно образование смесей окислителя с парами легковоспламеняющейся жидкости (нефти)
- При появлении источника зажигания возможно инициирование взрыва и (или) пожара
- Минимальная энергия зажигания составляет менее 0,25 мДж, что соответствует критериям пожаровзрывоопасности

## Заключение

В результате проведенного анализа и расчетов установлено:

1. Нефть устьевая относится к легковоспламеняющимся жидкостям (ЛВЖ) по классификации ГОСТ 12.1.044-2018
2. Показатели пожаровзрывоопасности:
   - Температура вспышки: 35°C
   - НКПР: 1,4% (об.)
   - ВКПР: 4,48% (об.)
   - Температура самовоспламенения: 480°C
   - Минимальная энергия зажигания: 0,157 мДж
   - Низшая теплота сгорания: 43 500 кДж/кг

3. Технологическая среда отнесена к **пожаровзрывоопасной** группе (Группа 2) согласно статье 16 Федерального закона № 123-ФЗ

4. При проектировании и эксплуатации помещений, где обращается нефть устьевая, необходимо предусматривать меры по предотвращению образования взрывоопасных концентраций паров и исключению источников зажигания.
