# Расчет показателей пожаровзрывоопасности технологических сред помещения 3.66 (Склад хранения проб)

## 1. Общие сведения

**Помещение:** 3.66 - Склад хранения проб (в том числе устьевых проб нефти и газа)  
**Площадь:** 34,30 м²  
**Высота:** 2,7 м  
**Объем помещения:** 92,61 м³  
**Температура в помещении:** 15-18 °C (принимаем 18 °C для расчетов)  
**Кратность общеобменной вентиляции:** 4  
**Система пожарной сигнализации:** присутствует  
**Система порошкового пожаротушения:** присутствует

## 2. Состав технологических сред

В помещении находятся следующие вещества:
1. **Пробы сырой устьевой нефти** - 430 литров в ПЭТ таре по 5 литров
2. **Стабильный конденсат** - 170 литров в ПЭТ таре
3. **Углеводородные газы** - 170 литров в баллонах по 10 литров
4. **Пробоотборники с флюидами нефти (углеводородный газ)** - 1.8 литра

## 4. Расчет показателей пожаровзрывоопасности

### 4.1. Оценка пожароопасных свойств жидких веществ

#### 4.1. Устьевая нефть

Температура вспышки нефти составляет около 28°C, что ниже температуры хранения (18°C). Однако при нормальных условиях хранения в герметичной таре по 5 литров, испарение ограничено.

Рассчитаем возможную концентрацию паров при аварийной ситуации (разлив всей нефти):

Площадь возможного разлива: 34,30 м²  
Объем помещения: 92,61 м³

Для температуры 18°C давление насыщенных паров нефти незначительно. Принимаем, что при этой температуре концентрация паров будет намного ниже НКПВ.

#### 4.1.2. Стабильный конденсат

Температура вспышки конденсата около 20°C, что выше температуры хранения (18°C). При нормальных условиях хранения в герметичной таре пожароопасность ограничена.

### 4.2. Оценка взрывоопасности газов

#### 4.2.1. Углеводородные газы в баллонах

Объем газов: 170 литров (0,17 м³) при нормальных условиях

При аварийной ситуации (разгерметизация всех баллонов) возможен выброс газа в объем помещения.

Рассчитаем концентрацию при равномерном распределении:
- Объем газа: 0,17 м³
- Объем помещения: 92,61 м³
- Концентрация: (0,17 / 92,61) × 100% = 0,18%

Эта концентрация значительно ниже НКПВ (2,5%), что указывает на отсутствие взрывоопасной концентрации при нормальных условиях.

#### 4.2.2. Флюиды нефти в пробоотборниках

Объем: 1,8 литра (0,0018 м³)

При аварийной ситуации:
- Концентрация: (0,0018 / 92,61) × 100% = 0,002%

Также значительно ниже НКПВ.

### 4.3. Расчет массы горючих веществ в помещении

Общая масса горючих жидкостей:
- Нефть: 378,92 кг
- Конденсат: 127,5 кг
- Итого: 506,42 кг

Объем горючих газов:
- При нормальных условиях: 0,1718 м³

### 4.4. Оценка пожарной нагрузки

Пожарная нагрузка Q (МДж) определяется по формуле:
Q = Σ(G_i × Q_i)

Где:
- G_i - количество i-го материала пожарной нагрузки, кг
- Q_i - низшая теплота сгорания i-го материала, МДж/кг

Для нефти: 378,92 кг × 43,6 МДж/кг = 16 521,31 МДж
Для конденсата: 127,5 кг × 44,5 МДж/кг = 5 673,75 МДж

Общая пожарная нагрузка: 16 521,31 + 5 673,75 = 22 195,06 МДж

Удельная пожарная нагрузка:
g = Q / S = 22 195,06 / 34,30 = 647,09 МДж/м²

### 4.5. Оценка категории помещения по пожарной опасности

Согласно СП 12.13130.2009:
- При удельной пожарной нагрузке 181 МДж/м² < g ≤ 1400 МДж/м² помещение относится к категории В3
- При удельной пожарной нагрузке 1401 МДж/м² < g ≤ 2200 МДж/м² помещение относится к категории В2

Поскольку 647,09 МДж/м² находится в диапазоне 181-1400 МДж/м², помещение потенциально может быть отнесено к категории В3.

Однако необходимо учитывать, что:
1. Вещества хранятся в герметичной таре
2. Помещение имеет вентиляцию с кратностью 4
3. Имеется система пожаротушения
4. Температура хранения ниже температуры вспышки

### 4.6. Оценка категории по взрывопожарной опасности

Для категории А:
- Помещение относится к категории А, если расчетное избыточное давление взрыва превышает 5 кПа

Для категории Б:
- Помещение относится к категории Б, если расчетное избыточное давление взрыва превышает 5 кПа для пылей

При аварийной ситуации максимальная концентрация газов составляет 0,18%, что ниже НКПВ (2,5%), следовательно, взрывоопасная концентрация не образуется.

## 6. Заключение

На основе проведенного анализа:

1. **Пожаровзрывоопасные свойства веществ:**
   - Устьевая нефть: ЛВЖ, температура вспышки 28°C, НКПВ 1,4%, ВКПВ 7,6%
   - Стабильный конденсат: ЛВЖ, температура вспышки 20°C, НКПВ 1,2%, ВКПВ 8,0%
   - Углеводородные газы: НКПВ 2,5%, ВКПВ 15,0%

2. **Классификация технологических сред:**
   - Нефть: пожароопасная среда (группа 1)
   - Конденсат: пожароопасная среда (группа 1)
   - Газы: пожаровзрывоопасная среда (группа 2)

3. **Оценка категории помещения:**
   - При нормальных условиях хранения в герметичной таре и при температуре 18°C (ниже температуры вспышки) образование взрывоопасных концентраций маловероятно
   - Удельная пожарная нагрузка составляет 647,09 МДж/м², что соответствует категории В3 при условии равномерного распределения нагрузки
   - Однако с учетом мер безопасности (вентиляция, пожаротушение, герметичная тара), помещение может быть отнесено к категории В1 при условии выполнения дополнительных требований

4. **Рекомендации:**
   - Обеспечить герметичность тары для хранения легковоспламеняющихся жидкостей
   - Поддерживать работоспособность системы вентиляции с кратностью 4
   - Обеспечить функционирование системы пожаротушения
   - Проводить регулярный контроль за состоянием герметичности тары
   - Ограничить доступ в помещение только для персонала, выполняющего внос и вынос веществ
