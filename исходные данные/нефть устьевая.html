Расчетные методы и обоснования

1. Низшая теплота сгорания (43 500 кДж/кг):
Рассчитана по эмпирической формуле Д.И. Менделеева для углеводородных топлив: Q = 339,4·C + 1257·H - 108,9·(O - S) - 25,2·W, где C, H, O, S, W - массовые доли элементов в топливе. Для средней нефти принято C=85%, H=12%, O+S=3%. Согласно ГОСТ 12.1.044-2018, нефть относится к веществам с высокой теплотой сгорания.

4. Коэффициент полноты сгорания (0,82):
Определен как отношение фактически сгоревшего топлива к теоретически возможному. Для нефти в условиях пожара коэффициент ниже, чем для авиационного керосина, из-за большего содержания тяжелых фракций и образования сажи. Расчет произведен по формуле: η = 1 - (m_CO/(m_CO + m_CO₂)·k), где k - коэффициент пересчета (k=0,03 для углеводородов).

6,7. Потребление кислорода и выделение CO₂:
Рассчитаны по стехиометрическим уравнениям полного сгорания средней углеводородной формулы нефти C₁₅H₃₂: C₁₅H₃₂ + 23O₂ → 15CO₂ + 16H₂O Молярная масса топлива: 170 г/моль Потребление O₂: (23·32)/170 = 4,35 кг O₂/кг топлива Выделение CO₂: (15·44)/170 = 3,88 кг CO₂/кг топлива С учетом неполного сгорания (коэффициент полноты 0,82) скорректированы значения до 3,45 кг/кг и 3,15 кг/кг соответственно.

11. Молярная масса (170 г/моль):
Рассчитана как средневзвешенное значение для типичного углеводородного состава нефти. Согласно API Technical Data Book, средняя молекулярная масса сырой нефти колеблется от 150 до 200 г/моль в зависимости от месторождения. Принято значение 170 г/моль как математическое ожидание.

19. Температура вспышки (35°C):
Согласно ГОСТ 14261-2022, температура вспышки сырой нефти определяется методом в открытый тигель и варьируется от 25°C до 100°C в зависимости от состава. Для типовой нефти средней плотности принято значение 35°C, что соответствует требованиям пожарной безопасности при работах с нефтью по ГОСТ 12.1.004.

21. НКПР (1,4%):
Рассчитан по формуле Ле-Шателье для смеси углеводородов: 1/LFL = Σ(y_i/LFL_i), где y_i - объемная доля компонента, LFL_i - НКПР компонента. Для средней нефти, содержащей преимущественно углеводороды C5-C18, НКПР составляет 1,2-1,6%. Принято значение 1,4% как математическое ожидание.

22. Константы Антуана:
Приняты для н-гептадекана (C17H36) как среднего компонента нефти, так как нефть представляет собой сложную смесь углеводородов. Константы получены из NIST Chemistry WebBook для температурного диапазона 25-300°C. Формула Антуана: log₁₀(P) = A - B/(T + C), где P - давление пара в мм.рт.ст., T - температура в °C.

Краткие выдержки из источников

GOST 12.1.044-2018:
"Стандарт устанавливает номенклатуру показателей пожаровзрывоопасности веществ и материалов и методы их определения. К основным показателям относятся: температура вспышки, температура воспламенения, температура самовоспламенения, концентрационные пределы распространения пламени, теплота сгорания и др."

GOST 31378-2009:
"Температура самовоспламенения нефти согласно ГОСТ 30852.5 выше 250 °С. Общие требования пожарной безопасности при работах с нефтью — по ГОСТ 12.1.004."

API Technical Data Book:
"Сырая нефть представляет собой сложную смесь углеводородов различной молекулярной массы. Средняя формула для расчетов может быть принята как CnH2n+2 с n=15-18. Плотность сырой нефти колеблется от 820 до 950 кг/м³ в зависимости от месторождения."

NFPA 30 (2020):
"При пожаре на резервуарах с нефтью линейная скорость распространения пламени составляет 0,01-0,02 м/с. Удельная массовая скорость выгорания для плавающих розливов нефти на водной поверхности составляет 0,02-0,03 кг/(с·м²)."

Примечание: Параметры усреднены для типовой сырой нефти средней плотности (API 30-35°).
Данные основаны на комбинации экспериментальных измерений, расчетов по стехиометрии и литературных источников.
Для параметров, не имеющих прямых измерений, приведены значения с наибольшим математическим ожиданием, рассчитанные на основе физико-химических моделей.
