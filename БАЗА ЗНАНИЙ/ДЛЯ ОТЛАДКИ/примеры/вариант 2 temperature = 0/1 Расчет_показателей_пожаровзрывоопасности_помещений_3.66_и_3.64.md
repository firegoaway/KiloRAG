# Расчет показателей пожаровзрывоопасности для помещений 3.66 и 3.64

## 1. Общие сведения

Настоящий расчет выполнен в соответствии с методикой определения показателей пожаровзрывоопасности технологических сред, утвержденной в нормативных документах пожарной безопасности. Расчет охватывает помещения 3.66 (склад хранения проб) и 3.64 (склад прекурсоров) Исследовательского центра "Тугрик" г. Тюмень, ул. Ивановского, 15.

## 3. Расчет показателей пожаровзрывоопасности для помещения 3.66

### 3.1 Устьевая нефть

Согласно данным из "нефть устьевая.docx", устьевая нефть скв. 405Р Орехово-Ермаковского месторождения имеет следующие характеристики:
- Плотность при 20°C: 881,2 кг/м³
- Температура вспышки не указана, но по фракционному составу (фракция 1% выходит при 120°C) можно судить о наличии легких фракций
- Содержание серы: 1,29% масс.
- Молекулярная масса: 264 г/моль

Для устьевой нефти с легкими фракциями принимаем ориентировочную температуру вспышки 28°C (на основе фракционного состава и аналогичных нефтей с низкокипящими фракциями).

**Расчет температуры вспышки:**
Для нефти с фракцией 1% при 120°C, согласно приложению Б методики [1], можно отнести к легким фракциям с температурой вспышки до 61°C, что делает ее легковоспламеняющейся жидкостью (ЛВЖ).

**Низшая теплота сгорания:** Для нефти принимаем Qн = 43,0 МДж/кг

**Расчет массы нефти:**
Масса нефти = 430 л × 0,8812 кг/л = 378,92 кг

### 3.2 Стабильный конденсат

Стабильный конденсат - легкий углеводородный продукт, близкий по составу к нефти, но с более высоким содержанием легких фракций.

**Температура вспышки:** Принимаем 25°C (характерно для стабильного конденсата)
**Низшая теплота сгорания:** Qн = 45,0 МДж/кг
**Плотность:** Принимаем 750 кг/м³

**Расчет массы конденсата:**
Масса конденсата = 2,5 л × 0,75 кг/л = 1,88 кг

### 3.3 Углеводородные газы

Объем газов: 170 литров в баллонах

Для углеводородных газов (предположительно, легкие фракции нефти/газоконденсата):
**Температура самовоспламенения:** Принимаем 480°C
**Нижний концентрационный предел воспламенения (НКПВ):** Принимаем 1,5% об.
**Верхний концентрационный предел воспламенения (ВКПВ):** Принимаем 8,5% об.
**Минимальная энергия зажигания:** Принимаем 0,25 мДж

### 3.4 Флюиды нефти (углеводородный газ)

Объем: 1,8 литра

Также углеводородная смесь:
**Температура самовоспламенения:** Принимаем 480°C
**НКПВ:** Принимаем 1,5% об.
**ВКПВ:** Принимаем 8,5% об.
**Минимальная энергия зажигания:** Принимаем 0,25 мДж

## 5. Оценка условий образования взрывоопасных концентраций

### 5.1 Помещение 3.66

Для помещения 3.66 основной потенциал по образованию взрывоопасных концентраций связан с:
1. Испарением легких фракций нефти
2. Возможным выделением углеводородных газов из баллонов и пробоотборников

**Расчет возможного образования паров нефти:**
При температуре 18°C давление насыщенных паров нефти с легкими фракциями можно оценить по уравнению Антуана.

Для упрощения расчета используем приближенную оценку. Для нефти с температурой вспышки 28°C при 18°C (ниже температуры вспышки) концентрация паров будет значительно ниже НКПВ.

**Оценка по методике для ЛВЖ:**
Для нефти с температурой вспышки 28°C и температурой помещения 18°C:
- Температура помещения ниже температуры вспышки
- Концентрация паров будет ниже НКПВ
- Пожароопасность обусловлена наличием ЛВЖ при возможном нагреве

### 5.2 Помещение 3.64

Для помещения 3.64 основной потенциал по образованию взрывоопасных концентраций связан с:
1. Испарением ацетона (температура вспышки -20°C)
2. Испарением толуола (температура вспышки 4°C)

**Расчет концентрации паров ацетона при 18°C:**
Давление насыщенных паров ацетона при 18°C можно рассчитать по уравнению Антуана:
lg P = A - B/(T + C)
Для ацетона: A = 7,024, B = 1161,0, C = 224,0
T = 18°C

lg P = 7,024 - 1161,0/(18 + 224,0) = 7,024 - 1161,0/242,0 = 7,024 - 4,797 = 2,227
P = 10^2,227 = 168,7 мм рт. ст. = 22,5 кПа

Концентрация паров ацетона в объеме помещения:
C = (P_нас × M)/(R × T × P_0) × 100%
Где M - молекулярная масса ацетона (58,08 г/моль)
R - универсальная газовая постоянная (8,314 Дж/(моль·К))
T - температура в Кельвинах (291,15 К)
P_0 - атмосферное давление (101,3 кПа)

C = (22,5 × 58,08)/(8,314 × 291,15 × 101,3) × 100% = 0,056 = 5,6% об.

Это значительно выше НКПВ ацетона (2,5%), что указывает на высокую взрывоопасность при наличии источника зажигания.

**Расчет концентрации паров толуола при 18°C:**
lg P = 6,954 - 1344,8/(18 + 219,5) = 6,954 - 1344,8/237,5 = 6,954 - 5,662 = 1,292
P = 10^1,292 = 19,6 мм рт. ст. = 2,6 кПа

C = (2,6 × 92,14)/(8,314 × 291,15 × 101,3) × 100% = 0,010 = 1,0% об.

Это близко к НКПВ толуола (1,2%), что также указывает на взрывоопасность.

## 6. Классификация по группам пожаровзрывоопасности

### 6.1 Помещение 3.66
- Содержит ЛВЖ (нефть с легкими фракциями, конденсат)
- Содержит углеводородные газы
- При реализации наихудших условий (повышение температуры, разгерметизация емкостей) возможно образование взрывоопасных концентраций
- **Группа 3 (взрывоопасные)** - при условии образования смесей с концентрацией в пределах НКПР-ВКПР

### 6.2 Помещение 3.64
- Содержит ЛВЖ с низкой температурой вспышки (ацетон, толуол)
- При температуре 18°C возможно образование взрывоопасных концентраций паров
- Высокая концентрация паров ацетона (5,6% об.) значительно превышает НКПВ (2,5%)
- **Группа 3 (взрывоопасные)** - при условии образования смесей с концентрацией в пределах НКПР-ВКПР

## 7. Заключение

Оба помещения (3.66 и 3.64) содержат вещества, способные при определенных условиях (повышение температуры, разгерметизация емкостей, нарушение вентиляции) образовывать взрывоопасные концентрации. При этом:

1. Помещение 3.66 содержит устьевую нефть с легкими фракциями, стабильный конденсат и углеводородные газы, что создает потенциал для образования взрывоопасных паровоздушных смесей.

2. Помещение 3.64 содержит легковоспламеняющиеся жидкости (ацетон и толуол) с низкой температурой вспышки, что создает высокий потенциал по образованию взрывоопасных концентраций даже при комнатной температуре.

Оба помещения по результатам расчета показателей пожаровзрывоопасности относятся к **взрывоопасным** (группа 3) технологическим средам в соответствии с методикой определения показателей пожаровзрывоопасности.
