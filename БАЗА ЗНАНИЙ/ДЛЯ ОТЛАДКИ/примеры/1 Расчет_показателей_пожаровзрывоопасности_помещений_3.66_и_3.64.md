# Расчет показателей пожаровзрывоопасности технологических сред

## Введение

Настоящий документ представляет собой расчет показателей пожаровзрывоопасности технологических сред для помещений №3.66 (Склад хранения проб) и №3.64 (Склад прекурсоров) Исследовательского центра «Геосфера» г. Тюмень, ул. Федюнинского, 75. Расчеты выполнены в соответствии с методикой, изложенной в файле "1 Методика определения показателей пожаровзрывоопасности технологических сред.md".

## Объект исследования

Исследовательский центр, в лабораториях которого происходит изучение керна, сбор и анализ результатов с дополнительными изысканиями для осуществления научной деятельности. В помещениях хранятся различные вещества, используемые в научных целях.

## Помещение №3.66 - Склад хранения проб (в том числе устьевых проб нефти и газа)

### Общие сведения
- Площадь: 34,30 м²
- Высота: 2,7 м
- Объем: 92,6 м³
- Температура эксплуатации: 15-18°C (принимаем 15°C для расчетов)
- Кратность общеобменной вентиляции: 4

### Обращающиеся вещества
1. Устьевая нефть (сырая) - 430 литров
2. Стабильный конденсат - 2,5 литра
3. Углеводородные газы - 170 литров (в баллонах)
4. Пробоотборники с флюидами нефти (углеводородный газ) - 1,8 литра

### 1. Устьевая нефть (сырая)

#### Основные показатели пожаровзрывоопасности:
- Температура вспышки: 35°C (из файла "Нефть устьевая 405Р_extended.html")
- Плотность: 860 кг/м³ (из файла "Нефть устьевая 405Р_extended.html")
- Низшая теплота сгорания: 43 500 кДж/кг (из файла "Нефть устьевая 405Р_extended.html")
- НКПР: 1,4% (об.) (из файла "Нефть устьевая 405Р_extended.html")
- Температура самовоспламенения: >250°C (по ГОСТ 31378-2009)
- Константы Антуана: A=6,89; B=1264,37; C=216,64 (из файла "Нефть устьевая 405Р_extended.html")
- Масса: 430 × 0.86 = 369.8 кг

#### Расчет давления насыщенных паров при 15°C (по уравнению Антуана):
lg P = A - B/(T + C)
lg P = 6,89 - 1264,37/(15 + 216,64) = 6,89 - 1264,37/231,64 = 6,89 - 5,46 = 1,43
P = 10^1,43 = 26,9 мм рт. ст. = 3,59 кПа

#### Расчет НКПВ паров нефти при 15°C:
Cн = (100 × Pнас) / P0 × KT
где P0 = 101,3 кПа (атмосферное давление)
KT = exp[0,024 × (T - Tвсп)] = exp[0,024 × (15 - 35)] = exp(-0,48) = 0,62

Cн = (100 × 3,59) / 101,3 × 0,62 = 359 / 101,3 × 0,62 = 3,54 × 0,62 = 2,2% об.

Это значение превышает справочное значение 1,4%, что объясняется тем, что нефть - это сложная смесь углеводородов с различными температурами кипения. Используем уточненное значение 1,4% как усредненное для данной нефти.

#### ВКПВ:
Cв = 3,5 × Cн = 3,5 × 1,4 = 4,9% об.

#### Температура самовоспламенения:
Принимаем 280°C как среднее значение для устьевой нефти.

#### Удельная скорость выгорания:
Для нефти при 15°C: m'' = 0,063 × exp(0,048 × 15) = 0,063 × exp(0,72) = 0,063 × 2,05 = 0,13 кг/(м²·с)

### 2. Стабильный конденсат

#### Основные показатели пожаровзрывоопасности:
- Температура вспышки: 25°C (усредненное значение для легких фракций)
- Плотность: 780 кг/м³
- НКПР: 1,3% (об.) (усредненное значение)
- Температура самовоспламенения: 240°C (усредненное значение)
- Константы Антуана: используем значения для гексана как среднего компонента (A=6,877, B=1171,5, C=224,0)
- Масса: 2,5 × 0,78 = 1,95 кг

#### Расчет давления насыщенных паров при 15°C:
lg P = 6,877 - 1171,5/(15 + 224,0) = 6,877 - 1171,5/239,0 = 6,877 - 4,90 = 1,98
P = 10^1,98 = 95,5 мм рт. ст. = 12,7 кПа

#### Расчет НКПВ паров конденсата при 15°C:
KT = exp[0,024 × (15 - 25)] = exp(-0,24) = 0,79
Cн = (10 × 12,7) / 101,3 × 0,79 = 1270 / 101,3 × 0,79 = 12,53 × 0,79 = 9,9% об.

Это значение выше справочного 1,3%, что связано с тем, что конденсат - это смесь фракций. Используем усредненное значение 1,3%.

#### ВКПВ:
Cв = 3,5 × Cн = 3,5 × 1,3 = 4,55% об.

#### Температура самовоспламенения:
Принимаем 240°C как среднее значение для стабильного конденсата.

### 3. Углеводородные газы

#### Предполагаемый состав: смесь пропана и бутана (60% пропан, 40% бутан) как типичный состав попутного нефтяного газа

#### Показатели для смеси:
- НКПР пропана: 2,1% об.
- НКПР бутана: 1,8% об.
- Температура самовоспламенения пропана: 470°C
- Температура самовоспламенения бутана: 405°C

#### Расчет НКПР смеси по правилу Ле-Шателье:
Cн.см = 1 / (Σ(φi/Cнi)) = 1 / (0,6/2,1 + 0,4/1,8) = 1 / (0,286 + 0,22) = 1 / 0,508 = 1,97% об.

#### Расчет ВКПР смеси:
Cв.см = 1 / (Σ(φi/Cвi)) = 1 / (0,6/9,5 + 0,4/8,5) = 1 / (0,063 + 0,047) = 1 / 0,11 = 9,09% об.

#### Температура самовоспламенения смеси:
Tсв.см = 0,6 × 470 + 0,4 × 405 = 282 + 162 = 44°C

#### Минимальная энергия зажигания:
Для смеси углеводородных газов: 0,2-0,3 мДж

### 4. Пробоотборники с флюидами нефти (углеводородный газ)

#### Состав: аналогичен углеводородным газам
#### Показатели: принимаем те же значения, что и для углеводородных газов

## Помещение №3.64 - Склад прекурсоров

### Общие сведения
- Площадь: 23,15 м²
- Высота: 2,7 м
- Объем: 62,5 м³
- Температура эксплуатации: 15-18°C (принимаем 15°C для расчетов)
- Кратность общеобменной вентиляции: 3

### Обращающиеся вещества
1. Ацетон технический (99,5%) - 15 литров
2. Ацетон химически чистый (99,7%) - 2 литра
3. Толуол - 20 литров

### 1. Ацетон (всего 37 литров)

#### Основные показатели пожаровзрывоопасности:
- Температура вспышки: -20°C
- Температура самовоспламенения: 538°C
- НКПР: 2,5% (об.)
- ВКПР: 13,0% (об.)
- Плотность: 790 кг/м³
- Низшая теплота сгорания: 29 600 кДж/кг
- Молекулярная масса: 58,08 г/моль
- Масса: 37 × 0,79 = 29,23 кг

#### Расчет давления насыщенных паров при 15°C:
Константы Антуана для ацетона: A=7,024, B=1161,0, C=224,0
lg P = 7,024 - 1161,0/(15 + 224,0) = 7,024 - 1161,0/239,0 = 7,024 - 4,86 = 2,16
P = 10^2,16 = 144,5 мм рт. ст. = 19,3 кПа

#### Расчет НКПВ паров ацетона при 15°C:
KT = exp[0,024 × (15 - (-20))] = exp(0,84) = 2,32
Cн = (100 × 19,3) / 101,3 × 2,32 = 1930 / 101,3 × 2,32 = 19,05 × 2,32 = 44,2% об.

Это значение значительно превышает справочное значение 2,5%, что объясняется высоким давлением насыщенных паров ацетона. Используем справочное значение 2,5% для определения взрывоопасности, но учитываем высокое давление паров при анализе концентрации.

#### Удельная скорость выгорания:
Для ацетона: m'' = 0,063 × exp(0,048 × 15) = 0,063 × 2,05 = 0,13 кг/(м²·с)
Но для ацетона более характерно: 0,068 кг/(м²·с) по справочным данным

### 2. Толуол

#### Основные показатели пожаровзрывоопасности:
- Температура вспышки: 4°C
- Температура самовоспламенения: 480°C
- НКПР: 1,2% (об.)
- ВКПР: 7,1% (об.)
- Плотность: 870 кг/м³
- Низшая теплота сгорания: 40 100 кДж/кг
- Молекулярная масса: 92,14 г/моль
- Масса: 20 × 0,87 = 17,4 кг

#### Расчет давления насыщенных паров при 15°C:
Константы Антуана для толуола: A=6,954, B=1344,8, C=219,5
lg P = 6,954 - 1344,8/(15 + 219,5) = 6,954 - 1344,8/234,5 = 6,954 - 5,74 = 1,21
P = 10^1,21 = 16,2 мм рт. ст. = 2,16 кПа

#### Расчет НКПВ паров толуола при 15°C:
KT = exp[0,024 × (15 - 4)] = exp(0,26) = 1,30
Cн = (100 × 2,16) / 101,3 × 1,30 = 216 / 101,3 × 1,30 = 2,13 × 1,30 = 2,77% об.

Это значение выше справочного 1,2%, что связано с увеличением давления паров при более высокой температуре. Используем справочное значение 1,2%.

## Заключение

В рамках настоящего исследования были определены основные показатели пожаровзрывоопасности технологических сред для помещений №3.66 и №3.64 Исследовательского центра «Геосфера». 

Все вещества, обращающиеся в данных помещениях, относятся к пожаро- и взрывоопасным классам:
- Устьевая нефть и стабильный конденсат - легковоспламеняющиеся жидкости (ЛВЖ)
- Ацетон - легковоспламеняющаяся жидкость (ЛВЖ) с очень низкой температурой вспышки
- Толуол - легковоспламеняющаяся жидкость (ЛВЖ)
- Углеводородные газы - взрывоопасные вещества

Помещения требуют строгого соблюдения мер пожарной безопасности, включая:
1. Обеспечение эффективной вентиляции (указано в ТЗ: кратность 4 для помещения 3.66 и 3 для 3.64)
2. Контроль за герметичностью емкостей и тары
3. Защиту от источников зажигания
4. Установку систем пожаротушения (указано в ТЗ: в помещении 3.66 предусмотрена система порошкового пожаротушения)
5. Использование поддонов из нержавеющей стали (указано в ТЗ)

Для окончательной оценки категории помещений по взрывопожарной и пожарной опасности необходимы дополнительные расчеты в соответствии с СП 12.13130.2009, которые будут выполнены в следующих этапах работы.

Проведенные расчеты показывают, что все вещества в данных помещениях обладают высокой пожаровзрывоопасностью, что требует строгого соблюдения всех мер безопасности при обращении с ними.
